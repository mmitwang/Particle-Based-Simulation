# 粒子法岩土边坡仿真软件

基于粒子法（SPH/PFC）实现的岩土边坡稳定性仿真分析软件，可用于边坡工程设计、稳定性评估和灾害预警。

## 主要功能

- **边坡建模**：参数化构建边坡模型，支持不同地貌和天气条件
- **仿真控制**：设置仿真参数，支持CPU/GPU加速
- **结果分析**：查看稳定性指标、位移场和应力应变场
- **拓展功能**：参数模板管理、批量仿真

## 技术架构

- **核心层**：包含粒子系统、邻居搜索、本构模型等核心算法
- **功能层**：包含边坡模型构建、仿真控制、稳定性评估等功能模块
- **UI层**：使用PyQt5实现主窗口和各种功能界面

## 依赖库

- PyQt5：GUI开发
- NumPy：数值计算
- Matplotlib：数据可视化
- SciPy：科学计算
- CuPy（可选）：GPU加速

## 使用说明

1. 运行 `python main_app.py` 启动软件
2. 在**边坡建模**页面设置模型参数，点击"构建边坡模型"
3. 在**仿真控制**页面设置仿真参数，点击"开始仿真"
4. 在**结果分析**页面查看仿真结果
5. 可选：在**拓展功能**页面保存参数模板或进行批量仿真

## 项目结构

- `main_app.py`：主应用程序文件
- `run_simulation.py`：核心仿真功能模块
- `generate_manual.py`：生成使用说明书
- `README.md`：项目说明文档
